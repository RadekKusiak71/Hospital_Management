## Instalation:
  - Clone repository
  - Create virtual environment
    ```
    > virtualenv env
    ```
  - Activate virtual environment:
    ```
    env/Scripts/activate
    ```
  - Install packages
    ```
    pip install -r requirements.txt
    ```
  - Start Server :)
    ```
    py manage.py runserver
    ```
  - Test User
    ```
    Login : admin
    Password: 123
    ```
## About Project
