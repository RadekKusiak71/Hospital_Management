# Hospital_Management still in development
## Technologies used:
  - Django
  - Python
  - Html
  - Css
  - Bootstrap
  - Django Rest Framework
## Instalation:
  - Clone repository
  - Create virtual environment
    ```
    > virtualenv env
    ```
  - Activate virtual environment:
    ```
    env/Scripts/activate
    ```
  - Install packages
    ```
    pip install -r requirements.txt
    ```
  - Start Server :)
    ```
    py manage.py runserver
    ```

    ```
    Login : admin
    Password: 123
    ```
